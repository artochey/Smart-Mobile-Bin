Smart Mobile Bin

คู่มือการใช้งานระบบถังขยะอัจฉริยะเคลื่อนที่ผ่าน Cloud (User Manual)
โครงการ: ถังขยะอัจฉริยะเคลื่อนที่ผ่าน Cloud (Smart Mobile Bin)
คณะผู้จัดทำ: VSEG TEAMS

1. บทนำ (Introduction)
คู่มือฉบับนี้จัดทำขึ้นเพื่อให้ผู้ใช้งานเข้าใจถึงขั้นตอนการติดตั้ง การควบคุม และการบำรุงรักษาระบบถังขยะอัจฉริยะ ซึ่งถูกพัฒนาขึ้นโดยใช้เทคโนโลยี IoT เพื่ออำนวยความสะดวกในการจัดการขยะภายในอาคาร โดยเฉพาะสำหรับผู้สูงอายุและผู้พิการ

2. รายละเอียดอุปกรณ์และสถาปัตยกรรม (System Specifications)
ก่อนเริ่มใช้งาน ผู้ใช้ควรทำความคุ้นเคยกับส่วนประกอบหลักของหุ่นยนต์ ดังนี้:

ชุดประมวลผลหลัก: บอร์ด ESP32 NodeMCU (รองรับการเชื่อมต่อ Wi-Fi)

ระบบขับเคลื่อน: มอเตอร์เกียร์ DC พร้อมชุดควบคุม L298N

ระบบตรวจจับระยะ: เซนเซอร์ Ultrasonic (HC-SR04) สำหรับเปิดฝาและเบรกฉุกเฉิน

ระบบกลไกฝาถัง: เซอร์โวมอเตอร์ (Servo SG90)

แหล่งพลังงาน: แบตเตอรี่ Li-ion 18650 (7.4V - 8.4V)

3. ขั้นตอนการเตรียมการก่อนใช้งาน (System Setup)
การติดตั้งแบตเตอรี่: ตรวจสอบแบตเตอรี่ 18650 ทั้ง 2 ก้อนว่ามีแรงดันไฟเพียงพอ และติดตั้งเข้ากับรางถ่านใต้ฐานรถ

การเชื่อมต่อเครือข่าย (Network Configuration):

ตั้งค่า Mobile Hotspot บนสมาร์ทโฟน โดยใช้ SSID และ Password ตามที่กำหนดไว้ในตัวโค้ด

เปิดสวิตช์ตัวเครื่อง รอให้ไฟสถานะบนบอร์ด ESP32 ติดนิ่ง เพื่อยืนยันการเชื่อมต่อ Cloud

การเชื่อมต่อแอปพลิเคชัน: เปิดแอปพลิเคชัน Blynk IoT บนสมาร์ทโฟน ตรวจสอบสถานะอุปกรณ์ว่าขึ้นเป็น "Online"

4. วิธีการควบคุมและฟังก์ชันการทำงาน (Operating Instructions)
ระบบสามารถทำงานได้ 2 รูปแบบหลัก ดังนี้:

4.1 การควบคุมการเคลื่อนที่ (Manual Cloud Control)

ใช้ชุดคำสั่ง Joystick บนหน้าจอแอปพลิเคชัน Blynk ในการบังคับทิศทาง (เดินหน้า, ถอยหลัง, เลี้ยวซ้าย/ขวา)

สัญญาณจะถูกส่งผ่าน Cloud ทำให้สามารถบังคับถังขยะได้จากทุกที่ที่มีอินเทอร์เน็ต

4.2 การทำงานของระบบอัตโนมัติ (Automation & Edge Logic)

ระบบเปิด-ปิดฝาอัตโนมัติ: เมื่อผู้ใช้เดินเข้าใกล้ถังขยะ เซนเซอร์ Ultrasonic จะตรวจจับระยะ หากอยู่ในระยะที่กำหนด ฝาถังจะเปิดออกเองโดยอัตโนมัติ และปิดลงเมื่อผู้ใช้เดินออกห่าง

ระบบเบรกฉุกเฉิน (Auto-Brake): ในขณะที่เคลื่อนที่หากเซนเซอร์ IR หรือ Ultrasonic ตรวจพบสิ่งกีดขวางในระยะประชิด (น้อยกว่า 10 ซม.) ระบบจะสั่งหยุดมอเตอร์ทันทีเพื่อป้องกันการชน

5. การติดตามผลและวิเคราะห์ข้อมูล (Data Monitoring)
ผู้ใช้สามารถติดตามสถานะของถังขยะผ่าน Dashboard ในแอปพลิเคชัน ได้แก่:

Status Monitor: แสดงปริมาณขยะภายในถัง (แจ้งเตือนเมื่อขยะเต็ม)

Usage Analytics: แสดงสถิติความถี่ในการใช้งานและช่วงเวลาที่มีการทิ้งขยะมากที่สุด เพื่อการจัดการรอบการจัดเก็บขยะที่มีประสิทธิภาพ

6. ข้อควรระวังและการบำรุงรักษา (Safety & Maintenance)
การทำความสะอาด: หลีกเลี่ยงการใช้น้ำฉีดล้างตัวถังโดยตรง เนื่องจากมีอุปกรณ์อิเล็กทรอนิกส์อยู่ภายใน ควรใช้ผ้าชุบน้ำบิดหมาดเช็ดทำความสะอาดภายนอก

การชาร์จไฟ: ไม่ควรปล่อยให้แบตเตอรี่หมดเกลี้ยง (Deep Discharge) เพราะอาจทำให้แบตเตอรี่เสื่อมสภาพ ควรชาร์จเมื่อระบบเริ่มเคลื่อนที่ช้าลง

สภาพแวดล้อม: ไม่ควรใช้งานในพื้นที่ที่มีความร้อนสูงจัด หรือพื้นที่ที่มีน้ำขัง

กลไก: ห้ามฝืนเปิดฝาถังด้วยมือในขณะที่ระบบทำงาน เพราะอาจทำให้เฟืองมอเตอร์เซอร์โวเสียหาย
